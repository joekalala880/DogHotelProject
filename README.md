# DogHotelProject

## Overview

DogHotelProject is a Java application developed as part of my coursework at Southern Maine Community College (SMCC). The project simulates the management of a dog boarding hotel by tracking guest information, handling check-in and check-out operations, calculating feeding requirements, and maintaining hotel records.

The project was designed to strengthen my understanding of object-oriented programming, file processing, and software development fundamentals.

## Features

- Read guest information from a data file
- Check in dogs to available rooms
- Check out dogs and update hotel occupancy
- Generate feeding schedules
- Calculate daily food requirements
- Create backup records
- Menu-driven user interface

## Technologies Used

- Java
- Apache NetBeans IDE
- File Input/Output (I/O)
- Object-Oriented Programming (OOP)

## Project Structure

- `DogHotelProject.java` – Main application and menu system
- `Hotel.java` – Hotel management operations
- `Guest.java` – Guest information and stay records
- `Dog.java` – Dog data and feeding calculations
- `Pet.java` – Base pet functionality
- `Date.java` – Date handling utilities
- `guests.txt` – Sample guest data file
